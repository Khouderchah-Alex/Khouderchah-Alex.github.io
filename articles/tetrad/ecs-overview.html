Entity Component System
Part 1 - Overview

What is it?

The Entity-Component-System―or ECS―pattern is a software architecture pattern that allows objects to be defined based on the components they contain. That is, objects are defined by composition (has-a relationships) as opposed to the inheritence that most object-oriented programmers are used to using.
