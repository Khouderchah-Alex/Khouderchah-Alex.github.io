About

Interests, skills, and hobbies

    My relationship with computers has been one of interminable fascination. Both the practical and theoretical aspects of computer science are enjoyable and immensely rewarding, and I believe that a holistic understanding of computers necessitates the study of both aspects.

Health benefits aside, exercise serves as a constant reminder that one's limits are imposed by the mind, and that great improvement and achievement demand a willingness to go through pain. It reminds me that sustained comfort is not a sign of success, but of stagnation.

    The creation and performance of music parallels the act of programming in the sense that both require one to be fastidious while still ensuring that the sum of all parts is well-integrated. By singing and playing the flute & saxophone, I have the pleasure of―much like programming―challenging myself while at the same time partaking in an activity that is inherently enjoyable.

  • The future is uncertain, but bright. I am always looking for opportunities to work on compelling projects, meet interesting people, and improve myself.

Tetrad Game Engine

3D game engine using an entity-component-system architecture

The Tetrad game engine is a work-in-progress 3D game engine built using the entity-component-system architecture. It uses OpenGL for rendering, and strives to remain platform-independent. Some notable features built from scratch are:

  • Entity-Component-System framework
  • 3D Rendering system using OpenGL
  • GUI system (using the rendering system)

The project is being developed and tested on Windows and Linux, but does not use any libraries that would exclude the possibility of MacOS support in the future.

w3wm

Tiling desktop manager for Windows

w3wm is a project dedicated to providing Windows 7, 8, and 10 with an experience similar to that of using i3 on Linux. Windows are automatically tiled, and users can use the keyboard to navigate between, move, resize, open, and close windows. Built from the ground up with native support for multi-monitor setups and DPI-aware processes, w3wm is a desktop manager fit for the modern user who potentially has multiple monitors of high resolution.

Reliable Data-Transfer Protocol

User-mode implementation of reliable data-transfer protocol

This is an implementation of a library that provides an interface similar to that of the Berkeley sockets API, with the library itself providing reliable data transfer over UDP. While there are certainly similarities between the implemented protocol and TCP, there are a number of differences (for example, TCP uses cumulative ACKs, whereas the implemented protocol uses ACKs in a manner similar to that of Selective Repeat).
